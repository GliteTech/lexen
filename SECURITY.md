# Security Policy

## Supported Versions

The supported release line is `lexen-v1`.

## Reporting Sensitive Issues

Do not open a public issue for suspected credential exposure, private data exposure, proprietary
source material, or any vulnerability in the release tooling.

Report sensitive issues to `info@glite.ai` with the subject prefix `SECURITY: lexEN`. Include:

* the affected file or artifact;
* the commit or release if known;
* a short description of the risk;
* steps to reproduce or verify the issue.

The maintainers will review the report, acknowledge receipt when possible, and coordinate a fix or
public advisory if one is required.
